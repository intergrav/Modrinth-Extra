Modrinth Extra

Thank you for installing Modrinth Extra! If you haven't already read the page, this extension has almost little-to-no features currently. The only current feature is extra animations, like fading and sliding.

However, if you would like to contribute and add your own features to the project, feel free to make a pull request on the GitHub repository! It would be greatly appreciated and would definitely help us grow this project.

Anyways, go try it out on Modrinth! Enjoy.
